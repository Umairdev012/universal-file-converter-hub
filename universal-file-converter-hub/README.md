# Universal File Converter Hub

A comprehensive file conversion and utility tool supporting image formats, PDF operations, QR code generation, and more.

## Features

- **Image Conversion**: Convert between multiple image formats
- **Image Compression**: Reduce image file sizes
- **Image Resizing**: Resize images to custom dimensions
- **PDF Tools**: PDF manipulation and conversion tools
- **QR Code Generator**: Generate QR codes from text or URLs
- **Password Generator**: Create secure random passwords

## Usage

Open `index.html` in a web browser to access the application.
